以下是这四个指标的物理意义以及基于 Python 的标准计算方法：

### 0.  前置准备（去直流偏置）
在进行任何时域或频域计算前，必须先消除传感器采集时产生的直流漂移（零点偏移）。

```python
import numpy as np
from scipy.stats import kurtosis
from scipy.signal import hilbert

# 假设 signal 是通过前文加载函数获取的 PU 数据集振动信号 (1D Array)
# fs = 64000
signal = signal - np.mean(signal)
N = len(signal)
```

### 1. RMS (Root Mean Square - 均方根)

**物理意义**：RMS 反映了振动信号的**整体能量水平**。当轴承发生严重磨损或整体劣化时，系统的振动能量会显著增加，RMS 值会随之飙升。它对早期极微小的局部故障不敏感，但非常适合作为设备整体健康状态的宏观指标。

**代码实现**：
```python
# 计算平方的平均值，再开根号 
rms_value = np.sqrt(np.mean(signal**2)) 
print(f"RMS 值: {rms_value:.4f}")
```

### 2. Kurtosis (峭度)

**物理意义**：峭度是无量纲的统计量，专门用于衡量信号波形中**是否存在明显的“尖峰”或“冲击”**。
- 正常轴承的振动接近高斯白噪声，其峭度值（Pearson定义）通常在 **3.0** 左右。
- 当轴承内圈、外圈或滚动体出现局部剥落（点蚀）时，滚动体每次碾过缺陷处都会产生极短促的高频冲击（Impulse）。这些冲击会使得信号的概率密度函数尾部变厚，峭度值会迅速增大（可能达到 5、10 甚至更高）。峭度是**早期故障极其敏感的指示器**。\

**代码实现**：
```python
# 使用 scipy.stats.kurtosis 计算
# fisher=False 表示使用 Pearson 定义（正常为 3），若 fisher=True 则正常为 0
kurt_value = kurtosis(signal, fisher=False)
print(f"峭度值: {kurt_value:.4f}")
```

### 3. FFT (Fast Fourier Transform - 快速傅里叶变换频谱)

**物理意义**：将时域信号转换为频域信号，展示不同频率成分的振幅大小。通过 FFT，可以直观地看到不平衡（1X转频）、不对中（2X转频）、机械松动或高频结构共振区。

**代码实现**：
```python
# 使用 rfft 处理实数信号，速度更快
frequencies = np.fft.rfftfreq(N, d=1/fs)
# 计算幅值，并进行除以 N 的归一化，使其具有真实的物理幅值意义
fft_spectrum = np.abs(np.fft.rfft(signal)) / N

# 实际应用中，由于 PU 数据采样率高达 64kHz，奈奎斯特频率为 32kHz
# 你通常只需要截取前面的低中频部分 (例如 0~5000 Hz) 进行观察
```

### 4. Envelope Analysis (包络分析)

**物理意义**：包络分析是滚动轴承故障诊断中**最强大、最核心的技术**。 当轴承发生局部故障时，冲击脉冲会激发出系统的高频固有共振（通常在数千Hz以上）。真正的故障特征频率（如外圈故障频率 BPFO，内圈故障频率 BPFI，通常只有几十到两三百Hz）会被“调制”在这些高频共振波上。 包络分析就像是“收音机的解调过程”：
1. 剥离掉高频共振的载波（解调）。
2. 提取出外面那层随低频故障冲击而起伏的“包络线”。
3. 对这根包络线做 FFT（包络谱），就能清晰地看到故障特征频率的谱线（BPFO/BPFI）。

**代码实现**（基于 Hilbert 变换的宽带包络）：
```python
# 步骤 A: 计算解析信号 (Analytic Signal) 和包络线 (Envelope)
analytic_signal = hilbert(signal)
envelope = np.abs(analytic_signal)

# 步骤 B: 去除包络线的直流偏置 (非常关键，否则包络谱在 0Hz 处会有一个极大的假峰)
envelope = envelope - np.mean(envelope)

# 步骤 C: 对包络线进行 FFT 获取包络谱 (Envelope Spectrum)
env_frequencies = np.fft.rfftfreq(N, d=1/fs)
env_spectrum = np.abs(np.fft.rfft(envelope)) / N
# 故障特征寻找：你可以在 env_spectrum 中寻找 BPFO/BPFI 对应的频率位置
```
