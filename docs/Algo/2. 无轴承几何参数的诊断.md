条件固定为：

- **已知转速**，且这个转速有置信度或不确定度；
- **没有轴承几何参数**，因此不能直接计算 BPFI、BPFO、BSF、FTF；
- 只使用 **峭度 / 谱峭度、包络解调、包络 FFT** 来诊断。

在这个前提下，最严谨的结论是：

**你可以诊断“是否存在轴承类冲击故障”，并可量化其强弱、稳定性和随时间的演化；但一般不能严格细化到内圈、外圈、滚动体或保持架。**  
因为 BPFI、BPFO、BSF、FTF 是轴频和轴承几何参数的函数；NI 明确说明这些特征频率/阶次依赖滚动体直径、节圆直径、接触角和滚动体数，并且实际值还会因滑移与理论值略有偏差。

## 1. 已知转速后，诊断变量应切换到 order 域

既然转速已知，最合理的做法不是只盯 Hz，而是把包络谱峰值转换成 **order**：

$o = f_{env} / f_r$

其中 $f_{env}$ 是包络 FFT 中的频率，$f_r$ 是轴频。这样做的意义是把速度变化的影响显式归一化；如果转速有小波动，相关影响也会直接反映到 order 容差里。对于转速不完全恒定的情况，经典 bearing 诊断教程也明确提到，小的速度波动有时需要先通过 order tracking 之类的方法补偿。

如果转速有相对不确定度 $e_r$，则 order 的相对不确定度可近似写成同量级：

$\Delta o / o \approx \Delta f_r / f_r = e_r$

所以后面所有“谱线是否对齐”“谐波是否成立”的判断，都不该用一个固定频点，而应使用一个随 $o$ 扩大的容差带。

## 2. 为什么不能直接看原始 FFT

对滚动轴承局部故障，原始振动谱往往并不直接显示清晰的故障频率族。MathWorks 的示例明确展示了：对内圈故障，原始频谱在 BPFI 及其谐波处并没有清晰模式，而对外圈故障，包络谱如果不先选对带宽，也可能看不到 BPFO 的清晰峰列。与此同时，故障引起的是**冲击激发的高频共振，再被低频故障重复率调制**，因此包络信号比原始谱更有诊断信息。

所以在你的前提下，严谨路径应是：

**先找“最冲击”的共振带，再做包络，再对包络做 FFT。**

## 3. 推荐的严格计算流程

### 第一步：预处理

对原始振动信号做去均值、去趋势，并按固定窗长分段。窗长要足以覆盖多个转周期和多个潜在冲击重复周期，否则包络 FFT 的分辨率不够。这个属于信号处理常规要求。

### 第二步：用峭度或谱峭度选带

单个时域峭度只能告诉你“整体是否更冲击”，但不能告诉你**冲击主要落在哪个频带**。更有效的是使用 **kurtogram / spectral kurtosis** 在频带上局部计算峭度，从而定位“最高峭度”或“最高冲击信噪比”的带宽和中心频率。MathWorks 明确说明，kurtogram 和 spectral kurtosis 是为了找到“最高峭度的频带”，然后对这个频带做带通，以得到更具冲击性的信号供包络分析使用。

从实现上，可以记该最优频带为：

$B^* = [f_c - BW/2,; f_c + BW/2]$

这一步的输出不是故障类型，而是**最适合解调的共振带**。

### 第三步：带通滤波

对原始信号在 $B^*$ 上带通，得到：

$x_b(t)$

这样做的目的，是把“故障冲击激发的高频共振”保留下来，同时压制大量结构频率、齿频、工频及宽带背景。MathWorks 的 outer-race 示例显示，先经 kurtogram 指导的带通后，信号峭度会上升，包络谱中的故障峰列才明显显现出来。

### 第四步：包络解调

对 $x_b(t)$ 求包络，可用 Hilbert 解析信号法：

$z(t) = x_b(t) + j \mathcal{H}{x_b(t)}$

$e(t) = |z(t)|$

这里 $e(t)$ 是包络信号。对 bearing 局部故障而言，包络信号承载的是“冲击重复频率”的低频调制信息，因此更接近故障的重复率本身。MathWorks 明确指出，局部故障会调制相应 critical frequencies，包络信号包含了原始频谱中不容易看到的诊断信息。

### 第五步：对包络做 FFT

计算包络谱：

$E(f) = FFT{e(t)}$

然后转换到 order 域：

$E_o(o)$，其中 $o=f/f_r$

此时真正可用于诊断的对象，不再是原始谱，而是 **包络 FFT / 包络 order 谱**。

## 4. 在没有轴承几何时，包络 FFT 里要找什么

因为不能算 BPFI/BPFO/BSF/FTF，所以你不应该做“几何匹配”；你应该做的是寻找**bearing-like repetitive transients** 的数学特征：

### 4.1 重复冲击的基本峰列

若包络谱中出现某个非零基本 order $o_b$ 及其多个谐波：

$o_b,;2o_b,;3o_b,\dots$

并且这些峰在多个时间窗中稳定存在，那么可以严格说：  
**存在一个稳定的重复冲击过程，其重复率与轴转无锁定几何关系，但与滚动轴承局部缺陷的包络机理相容。**  
这一步并不能证明它是内圈还是外圈，但可以证明“信号里有规律冲击源，而不是纯随机噪声”。包络分析和 envelope power spectrum 正是用来观察这类 fault-frequency 峰列的。

### 4.2 高阶谐波的完整性

教程文献强调，bearing fault 的本质是脉冲序列；如果解调频带过窄，包络后只剩近似正弦，就会失去“冲击性”。因此，一个更可信的 bearing-like 故障，应在包络谱中呈现**多个谐波**而不是单峰。Randall 与 Antoni 的教程明确指出，要识别 bearing fault 的 impulsive nature，理想上需要覆盖足够多谐波；仅有很低阶信息时，故障冲击不容易被识别。

### 4.3 随时间的稳定性

真正的局部缺陷不会只在一个时间窗偶然出现一次峰列。更严格的判据应是：同一 $o_b$ 在相邻时间窗中持续存在，其谐波集合也基本稳定。  
因此可定义一个“峰列持续度”指标，例如统计在 $N$ 个时间窗中，有多少个时间窗在 $ko_b$ 附近检测到显著峰。

### 4.4 与 1x/结构性问题的可分离性

你前面讨论的 1x、2x、3x 多用于结构性振动问题；bearing fault 则更典型地表现为 **高频带中的冲击，经解调后在低频形成重复率峰列**。所以如果一个可疑峰族只在原始低频谱明显，而在最佳解调带的包络谱中不成立，它更可能是结构性问题而不是 bearing local fault。MathWorks 的示例也说明，raw spectrum 不足以诊断 bearing fault，而 envelope spectrum 才是关键。

## 5. 峭度、包络解调、FFT 各自承担什么角色

### 峭度 / 谱峭度

作用是：**发现“哪里最冲击”**。  
它回答的是频带选择问题，不直接回答 fault type。谱峭度或 kurtogram 的价值在于把最能体现重复瞬态的频带找出来。

### 包络解调

作用是：**把高频共振载波上的幅值调制取出来**。  
它把“高频振铃”变成“低频重复率”，从而让故障冲击频率出现在包络谱里。

### 包络 FFT

作用是：**把重复冲击变成可量化的谱线与谐波族**。  
它回答的是“是否存在稳定冲击重复率”以及“这个重复率的谐波结构是否自洽”。

三者组合起来，才是完整的 bearing-fault 检测链。

## 6. 在你的前提下，严格可下的诊断结论

如果满足下面四条：

1. 在某个由谱峭度选出的共振带上，带通信号峭度明显高于原始信号；
    
2. 包络 FFT 中出现稳定的基本 order $o_b$；
    
3. $o_b$ 的多个谐波在多个时间窗中持续存在；
    
4. 这一现象在不同工况或重复采集中可复现；
    

那么你可以严格得出：

**存在 bearing-related impulsive fault 的强证据。**

这里“bearing-related”是严格的，因为这一整套机理——局部接触缺陷引发冲击、高频共振、包络后出现重复率峰列——正是滚动轴承局部故障诊断的经典路径。MathWorks、NI 和 bearing diagnostics 教程都把包络谱作为局部缺陷频率识别的核心工具。

但在**没有几何参数**时，通常仍不能严格推出：

- 这是 BPFI；
    
- 这是 BPFO；
    
- 这是 BSF；
    
- 这是 FTF。
    

因为这些量本来就要靠轴承几何或厂家给出的 fault-frequency 数据来定义。NI 明确说明 characteristic orders 是几何参数的函数；Siemens 也明确说，进行 envelope curve analysis 需要 bearing geometry 或厂家 fault frequencies，且不同厂家同型号轴承的 fault frequencies 还可能不同。

## 7. 已知转速的置信度，应如何进入计算

既然你说“此转速有置信率”，那最合理的做法是把它转成**order 容差**。

若轴频估计为 $f_r$，相对不确定度为 $e_r$，则对包络谱中任一候选峰 $f_p$，其 order 为：

$o_p = f_p / f_r$

对应容差可取：

$\Delta o_p \approx o_p e_r$

于是谐波匹配不再是检查“是否恰好落在 $k o_b$”，而是检查是否落在：

$[k o_b - \Delta_k,; k o_b + \Delta_k]$

其中 $\Delta_k$ 至少应包含速度不确定度和 FFT 分辨率两部分。这样可以把“速度有置信度”直接融进数学判别，而不是把它当口头信息。

## 8. 一个可直接实现的判别流程

你可以按下面的顺序实现：

1. 从振动信号计算谱峭度 / kurtogram，得到最优解调带 $B^*$；
    
2. 在 $B^*$ 上带通，得到 $x_b(t)$；
    
3. 计算包络 $e(t)$；
    
4. 对 $e(t)$ 做 FFT，得到包络谱；
    
5. 用已知轴频把包络谱转换为 order 谱；
    
6. 在 order 谱中寻找“基本 order + 多个谐波”的 comb；
    
7. 计算三个指标：
    
    - 峭度增益；
        
    - 谐波数量与总能量；
        
    - 多窗持续度；
        
8. 若三者同时达到阈值，则判为 bearing-related impulsive fault；
    
9. 若只满足其中一部分，则只能判为可疑冲击异常；
    
10. 不做 BPFI/BPFO/BSF/FTF 细分，除非后续补充轴承几何或厂家 reference-speed fault frequencies。Siemens 的文档也说明，如果厂家直接给了 reference-speed fault frequencies，就可以按当前速度缩放使用，此时即使没有几何，仍可做精确匹配。
    

## 9. 最后压缩成一句话

在“**已知转速、未知轴承几何**”的条件下，  
**峭度/谱峭度负责选出最冲击的共振带，包络解调负责把高频冲击变成低频重复率，包络 FFT 负责证明是否存在稳定的 bearing-like 冲击峰列；因此可以严格诊断‘是否存在轴承局部冲击故障’，但一般不能严格细化到 BPFI、BPFO、BSF、FTF。**
