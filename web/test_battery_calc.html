参数 值 说明

ESP32工作电流 30 mA ESP32工作时的典型电流
4. 计算公式
步骤 A：计算一个周期内的总耗能 (Q_cycle，单位：毫安秒 mAs)
1. 采集次数 (N) = F_r / F_d
2. 采集总耗能 = N × t_s × I_sample
3. 上报总耗能 = 1 × t_r × I_report
4. 休眠总耗时 = F_r - (N × t_s) - t_r
5. 休眠总耗能 = 休眠总耗时 × I_sleep
6. 单周期总耗能: Q_cycle = (N × t_s × I_sample) + (t_r × I_report) + [F_r - (N × t_s) - t_r] × I_sleep
步骤 B：计算平均电流 (I_avg，单位：mA)
I_avg = Q_cycle / F_r
步骤 C：计算理论可用时间
可用小时数: Hours = C_eff / I_avg
可用天数: Days = Hours / 24
5. 测试用例：采集频率1小时，上报频率8小时（4G模式）
输入参数：
- 采集间隔 (F_d): 1小时 = 3600秒
- 上报间隔 (F_r): 8小时 = 28800秒
- 通讯方式: 4G (LTE)
电流计算：
- 休眠电流 (I_sleep): wifi.standby_deep_sleep = 0.01 mA
- 采集电流 (I_sample): ESP32(30) + imu.working(1.0) = 31 mA
- 上报电流 (I_report): ESP32(30) + imu.working(1.0) + cellular.working_avg(500.0) = 531 mA
步骤 A：计算总耗能
1. 采集次数 (N): F_r / F_d = 28800 / 3600 = 8次
2. 采集总耗能: 8 × 2 × 31 = 496 mAs
3. 上报总耗能: 1 × 20 × 531 = 10620 mAs
4. 休眠总耗时: 28800 - (8 × 2) - 20 = 28764秒
5. 休眠总耗能: 28764 × 0.01 = 287.64 mAs
6. 周期总耗能 (Q_cycle): 496 + 10620 + 287.64 = 11403.64 mAs
步骤 B：计算平均电流
平均电流 (I_avg): 11403.64 / 28800 = 0.396 mA
步骤 C：计算续航时间
有效容量 (C_eff): 9000 × 0.85 = 7650 mAh
可用小时数: 7650 / 0.396 = 19318.18小时
可用天数: 19318.18 / 24 = 804.92天
约: 2.2年
计算结果：
✓ 理论待机时长：约 805天 (2.2年)
与用户提供的示例计算结果一致。
6. JavaScript 实现验证
重新实现的 calculateBatteryLife 函数关键逻辑：
// 1. 定义变量
const C = batteryCapacity; // 9000 mAh
const η = 0.85;
const C_eff = C * η; // 7650 mAh

// 2. 电流参数
const I_sleep = powerConsumption.wifi_standby_deep; // 0.01 mA
const I_sample = 30 + powerConsumption.imu_working; // 31 mA
const I_report = commType === 1 
  ? 30 + powerConsumption.imu_working + powerConsumption.cellular_working // 531 mA (4G)
  : 30 + powerConsumption.imu_working + powerConsumption.wifi_working_tx; // 316 mA (WiFi)

// 3. 时间参数
const F_d = detectInterval * 60; // 秒
const F_r = detectInterval * reportCycle * 60; // 秒
const t_s = 2; // 秒
const t_r = 20; // 秒

// 4. 计算公式
const N = Math.floor(F_r / F_d);
const Q_sample = N * t_s * I_sample;
const Q_report = t_r * I_report;
const sleepSeconds = F_r - (N * t_s) - t_r;
const Q_sleep = sleepSeconds * I_sleep;
const Q_cycle = Q_sample + Q_report + Q_sleep;
const I_avg = Q_cycle / F_r;
const lifeHours = C_eff / I_avg;
const lifeDays = lifeHours / 24;
